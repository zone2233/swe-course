# Mission

## Project Name

Console ToDo App

## Mission Statement

The mission of console ToDo app is to allow users to manage tasks quickly without needing a graphical interface or external tools, through a simple terminal interface.

## Target Users

- Students
- People who want a simple local todo list
- Users who are comfortable using the terminal

## Problem Statement

Many todo apps have too many functions, are too complex and forget that their goal is actually very simple: tracking tasks. This project aims to solve this problem by providing a lightweight terminal app where users can add, view, complete, and delete tasks.

## Project Goals

- Provide a simple menu-based terminal interface.
- Allow users to add tasks.
- Allow users to view all tasks.
- Allow users to mark tasks as done.
- Allow users to delete tasks.
- Handle invalid input without crashing.
- Keep the program easy to understand and maintain.

## Future Implementations

The current version does not include, but they could be added in future versions:

- Saving tasks after the program exits
- User accounts
- Cloud storage
- A graphical user interface
- Task deadlines or categories

## Success Criteria

The project is successful if the user can:

- Start the program and understand the menu.
- Add a valid task.
- See all current tasks.
- Mark a task as done.
- Delete a task.
- Exit the program.
- Receive error messages for invalid input.
