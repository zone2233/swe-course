# Console ToDo Manager

## Description

This is a simple ToDo App in Python, where you can manage tasks.

## Features

- Add a new task
- Show all tasks
- Mark a task as done
- Delete a task
- Reject empty task titles

## How to Run

python3 src/main.py

## Personal Choice

I chose to do a ToDo app, because it is simple to implement and it is easy to further build improvements like adding file storage or tests. 

Furthermore, a ToDo app is useful for later tasks as well.
